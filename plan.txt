## Tools (4 Handlers)

### Tool 1: Token Safety Check
Input:  "Is 0xABC... safe?" / "Check token 0xABC..."
Flow:   Extract address -> TokenSafetyRegistry.getConsensus(addr)
        -> If no data: scan via GoPlus API -> write report on-chain
        -> Return: score, honeypot, mintable, taxes, verdict
Output: "Token 0xABC... scored 85/100 -- SAFE
         Honeypot: No | Mintable: No
         Buy Tax: 5% | Sell: 5%
         Consensus: 3 reporters agree"

### Tool 2: Yield Scanner
Input:  "Best yields?" / "Where to earn APY on Pharos?"
Flow:   YieldRegistry.getAllProtocols() -> for each: getLatestYield()
        -> Sort by APY, filter by risk level
Output: "TOP YIELDS ON PHAROS:
         1. ProtocolX PHRS/USDC -- APY: 45% -- TVL: $1.2M -- Risk: MEDIUM
         2. ProtocolY PHRS     -- APY: 12% -- TVL: $5M   -- Risk: LOW"

### Tool 3: Batch Safety Scan
Input:  "Scan top tokens" / "Which tokens are safe?"
Flow:   Fetch trending tokens -> batchIsTokenSafe(addresses[])
        -> Return grid
Output: "SAFETY SCAN RESULTS:
         [OK] Token A (0x123...) -- Score 92 -- SAFE
         [!!] Token B (0x456...) -- Score 45 -- CAUTION
         [XX] Token C (0x789...) -- Score 12 -- AVOID"

### Tool 4: Write Report (Agent -> Contract)
Input:  Internal trigger when scanning a new token
Flow:   GoPlus API -> parse safety data -> calculate score
        -> TokenSafetyRegistry.updateReport(token, score, ...)
        -> Update consensus
Output: "Report submitted for 0xABC... -- Score 85/100"

## Intent Parser

Simple keyword matching, no LLM:

  function parseIntent(message) {
    const lower = message.toLowerCase();
    const addressMatch = message.match(/0x[a-fA-F0-9]{40}/);
    const address = addressMatch ? addressMatch[0] : null;

    if (/safe|check|honeypot|rug|scam|token.*analyz/i.test(lower) && address) {
      return { tool: 'tokenSafety', args: { address } };
    }
    if (/yield|apy|earn|best.*pool|lend|stake/i.test(lower)) {
      const risk = /low.*risk|safe/i.test(lower) ? 1 :
                   /high.*risk|degen/i.test(lower) ? 3 : null;
      return { tool: 'yieldScan', args: { riskLevel: risk } };
    }
    if (/scan|top.*token|trending|which.*safe/i.test(lower)) {
      return { tool: 'batchScan', args: {} };
    }
    if (/wallet|portfolio|address.*0x/i.test(lower) && address) {
      return { tool: 'walletIntel', args: { address } };
    }
    return { tool: 'help', args: {} };
  }
